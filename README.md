# README

​	本仓库的内容为作者从事软件测试多年的一些个人总结，写这个仓库的目的1是因为目前网络上软件测试的知识都是散落到各个平台上，即使有一些测试人的优秀的平台，比如testerhome，但是上面的帖子也是各种解决问题贴、各种笔记贴，没有一个系统的学习贴，导致很多新手在职业过程中一般都是依赖于当前职位用到什么去学什么，即使一些大牛给出了学习路线，但是看到那个从组成原理到编程语言的思维导图，也可能吓的立马关闭了网页冷静冷静，所以就想已作者当前这点浅薄的一些知识和经验来给新手一些清晰一点、务实一点的东西，希望能为测试圈贡献一点微薄之力。目的2就偏个人了，最近作者正裸辞在家准备找工作中，发现一个惨痛的事实：【本人属于那种一做就会一面就废的人】，反思后觉得应该是因为平时只闷头做，总结也是一些零散的总结，没有形成自己的一套思维、行动体系，这样在面试的时候很吃亏，就会给面试官一种东一榔头西一榔头、条理不清晰的感觉，以上这两点促成了这个仓库的建立，希望先能把现有的知识梳理完成，以后能坚持补充完善，flag先立下来。
